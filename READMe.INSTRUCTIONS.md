This is a python coded wigit that codes servo movement for you by letting you select the pins you want to use as buttons and the servo.
to use the code you will need to install the library (reportlab) copy this line into your terminal in Vs Code     pip install reportlab  
once this library is installed you should be able to run the code, remember this code runs is Vs Code.
